# Resource Efficient Hybrid RAG:

This file includes the configurations needed to run a resource efficient RAG chatbot using a Small Language Model (SLM). Technologies and methodologies used:

1) Ollama: Configurations to run Ollama in a notebook / google colab
2) Small Language Model : Microsoft Phi
3) Hybrid search : Combines vector serach and knowledge graph
4) Chroma DB : For storing and retrieving embeddings
